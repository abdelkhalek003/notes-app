# NOTES APP

A simple notes-taking web app that allows creating, editing, and deleting notes with localStorage persistence.

## DEMO

Check out the live demo <[here]((https://prismatic-sopapillas-e8b770.netlify.app/))>.

## FEATURES

- Create, edit, and delete notes
- Persistent storage using localStorage
- toggle between light and dark themes

## TECHNOLOGIES USED

- HTML5
- CSS3
- JavaScript (ES6+)

## INSTALLATION

1. Clone the repository:
   git clone <[repository-link](https://github.com/abdelkhalek003/notes-app.git)>
2. Navigate to the project directory:
   cd notes-app
3. Open `index.html` in your web browser.

## USAGE

- Click on "Add Note" to create a new note.
- Click on a note to edit its content.
- Click the "Delete" button to remove a note.
- Notes are automatically saved to localStorage.
- click the theme toggle button to switch between light and dark modes.

## PROJECT STRUCTURE

- `index.html`: Main HTML file
- `styles.css`: CSS styles for the app
- `app.js`: JavaScript functionality for the app
